# วิธีการขอ CivitAI API Key

การที่มี CivitAI API Key จะช่วยให้การ download model จาก CivitAI ทำได้สะดวกขึ้น เช่น Download Model ที่ต้อง Login

## ขั้นตอนการขอ CivitAI API Key

1. กด Sign In
![](/civit_1_1.png)

2. เลือกวิธีการ Login เช่น Google, Discord, email ...

![](/civit_1_2.png)

3. เมื่อทำการ Login สำเร็จแล้ว ให้ไปที่รูป profile มุมขวาบนแล้วกด Account settings

![](/civit_1_3.png)

4. เมื่อเข้ามาแล้วให้เลื่อนลงไปข้างล่างสุดจนเจอ API Keys

![](/civit_1_4.png)

![](/civit_1_5.png)

5. กด Add API Key

![](/civit_1_6.png)

6. ตั้งชื่อ API Keys (1) แล้วกด save (2)

![](/civit_1_7.png)

7. เมื่อได้ API Keys แล้วให้ทำการ copy แล้วเก็บเอาไว้ (จะกลับมาดูใหม่ไม่ได้อีก หากหายต้องสร้างใหม่)

![](/civit_1_8.png)

เมื่อได้ CivitAI API Key แล้วให้ไปกรอกที่ช่อง `CIVITAI_API_KEYS` ได้เลย
